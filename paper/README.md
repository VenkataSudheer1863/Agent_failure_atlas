# Paper

This directory contains the full research paper draft for the Agent Failure Atlas project.

## Files

| File | Description |
|---|---|
| `paper.md` | Full paper draft in Markdown |
| `figures/` | All figures referenced in the paper (generated by analysis scripts) |
| `tables/` | LaTeX and Markdown versions of all tables |

## Generating Figures

```bash
# Generate all figures from the AFAD dataset
python dataset/generate_afad.py
python analysis/visualizations.py
# Or open analysis/failure_analysis.ipynb and run all cells
```

Figures are saved to `analysis/results/figures/` and should be copied to `paper/figures/`.
